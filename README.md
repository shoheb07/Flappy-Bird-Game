# Flappy-Bird-Game
Flappy Bird Clone is a simple 2D arcade-style game developed using Python and the Pygame library. The game replicates the basic mechanics of the classic Flappy Bird game, where the player controls a bird that must fly through gaps between moving pipes without colliding with them.

 Overview

Flappy Bird Clone is a simple 2D arcade game built using **Python and Pygame**.
The objective of the game is to control a bird and navigate it through gaps between moving pipes without hitting them.

The bird constantly falls due to gravity, and the player must press the **spacebar** to make the bird jump and avoid obstacles.



Features

* Simple and smooth gameplay
* Gravity-based bird movement
* Pipe obstacle generation
* Collision detection
* Score tracking
* 60 FPS game loop



 Technologies Used

* Python
* Pygame



 Installation

 1. Install Python

Download Python from the official website if it is not already installed.

 2. Install Pygame

pip install pygame

 3. Run the Game

Save the file as `flappy_bird.py` and run:


python flappy_bird.py


 Controls

| Key      | Action             |
| -------- | ------------------ |
| Spacebar | Make the bird jump |


 Game Objective

The objective is to fly the bird through the gaps between pipes without colliding with them. Each successful pass increases the player's score.
